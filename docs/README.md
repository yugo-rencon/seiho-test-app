# ドキュメント

このプロジェクトの設計・運用ドキュメントを `docs/` 配下にまとめます。

## 一覧
- `docs/ERD.md`  
  データベース設計（ER図・テーブル定義・リレーション）
- `docs/Architecture.md`  
  画面/ルーティング/主要機能の構成メモ
- `docs/Stripe.md`
  Stripeアカウント情報・運用メモ（秘匿情報は除く）
- `docs/Deploy.md`
  本番反映の手順（`scripts/deploy-prod.sh` の使い方）

## 方針
- 仕様変更が入ったら **該当ドキュメントも同時に更新**する
- 実装とズレた記述は残さない
